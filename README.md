- Tailwind CSS
- react-icon
- JSON Server (mock API)
- react-toastify

Usage

Install Dependencies
npm install

To Run JSON Server - http://localhost:8000/jobs
npm run server

To Run Vite Frontend - http://localhost:3000
npm run dev

Build for production
npm run build

Preview Production Build
npm run preview
